Frontend 101: HTML, CSS, JavaScript y Git

Una página que resumen los pilares del desarrollo web.

1 HTML

HTML (HyperText Markup Language) define la estructura y el significado del contenido en la web.

  • Etiquetas comunes: <h1>, <p>, <a>, <img>, <section>.
  • Documento base: <!doctype html>, <html>, <head>, <body>.
  • Semántica para accesibilidad y SEO: usa <header>, <main>, <article>, <footer>.
estructura

2 CSS

CSS (Cascading Style Sheets) controla la presentación: colores, tipografías, layout y responsividad.

  • Selección por etiqueta, clase o id: p, .boton, #header.
  • Modelos de diseño: flex, grid, position.
  • Buenas prácticas: variables, sistemas de espaciado, mobile-first y componentes reutilizables.
estilos

3 JavaScript

JavaScript agrega interactividad y lógica: escuchar eventos, modificar el DOM y conectar APIs.

  • DOM: document.querySelector(), addEventListener(), classList.toggle().
  • Módulos y fetch: import/export, fetch() para consumir servicios.
  • Buenas prácticas: código modular, funciones puras y manejo de errores.
interactividad

4 Git

Git es el sistema de control de versiones más usado: registra cambios y facilita la colaboración.

  • Flujo básico: git init → git add → git commit → git push.
  • Ramas: crea features en git branch y fusiona con git merge o git rebase.
  • Remotos: conecta con plataformas como GitHub, GitLab o Bitbucket.
