📖 Teoría de la Notifications API

La Notifications API permite mostrar notificaciones en el escritorio o dispositivo del usuario desde la web, previo permiso.

  • Notification.requestPermission(): solicita permiso al usuario para mostrar notificaciones.
  • new Notification(title, options): crea y muestra una notificación. options puede incluir body, icon, tag, etc.
  • Se puede controlar qué pasa al hacer click en la notificación con onclick.
  • Siempre verificar que Notification.permission sea 'granted' antes de mostrar la notificación.

En este playground podés probar cómo pedir permiso y enviar notificaciones con un log de seguimiento.
