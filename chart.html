Playground: Chart.js

Intercambia entre distintos tipos de gráficos para aprender cómo funciona la librería.

Explicación teórica

Chart.js es una librería de JavaScript que permite crear gráficos de forma sencilla usando el elemento <canvas>. Para usarla, seguimos estos pasos:

  1. Incluimos la librería (CDN o instalación local).
  2. Definimos un <canvas> en el HTML.
  3. Instanciamos un nuevo Chart en JS, indicando:
    • El tipo de gráfico (bar, line, pie, etc.).
    • Los datos (labels y datasets).
    • Las opciones de configuración (colores, títulos, leyendas, etc.).

Métodos clave:

  • new Chart(ctx, config): crea un gráfico en el contexto de un canvas.
  • chart.update(): actualiza el gráfico si cambian los datos.
  • chart.destroy(): destruye el gráfico actual, útil para crear uno nuevo.
